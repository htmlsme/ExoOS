# ExoOS - An operating system project

Its a hobby project so dont expect it to be insanely proficent, optimized, and totally bug free. (Or even expect it to be finished cuz I get burnt out easily sorry :/)

# Prerequisites

1. MUST use a linux machine or the Windows Subsytem for Linux (WSL)
2. MUST have 'nasm' installed
3. MUST have 'make' installed
4. MUST have the 'qemu-full' package installed (NOT 'qemu-base' idk if it will work)
5. Have a text editor (If you want to make a fork also DONT USE EMACS!!!)

# Compiling

1. In the project DIR run "make BUILD" in terminal

# Booting

 1. In the project DIR run "make BOOT" in terminal

# Clearing build DIR

1. In the project DIR run "make CLEAR" in terminal
